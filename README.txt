Projet My first chat bot
Lucas Monnais et Antonin Croquette

https://github.com/Antonain1/TF-IDF_project

Fonctionnalités :

	Analyse de la question avec la méthode TF-IDF
	Utilisation du chat bot pour répondre à une question
	Traitement des textes présents dans le dossier speeches avec la méthode TF-IDF
	Peut afficher la liste des mots les moins importants ou les mots les plus important du corpus de document
	Peut afficher les mots les plus répétés par Chirac
	Peut afficher les présidents qui ont parlé du mot "nation" ainsi que le discours où il est le plus présent (donc le président qui l'a le plus dit)
	Peut afficher quel président a parlé d'écologie
	Peut afficher quels mots ont été dit par tous les présidents

Instructions d'exécution :

	-ouvrir le dossier TF-IDF_project sur pycharm
	-exécuter le main.py
	-suivre les instructions sur la console
		Si vous choisissez le chat bot (1) : posez votre question sur l'un des président français, le chat bot vous répondra le plus précisément 						     possible.
		Si vous choisissez les fonctionnalités (2) : observez l'utilisation de la méthode TF-IDF pour analysez une base de données et obtenez des 							     informations précises en choisissant parmi les fonctionnalités.
